  Программа чтения и обработки информации из журнала системных событий
 Windows NT. Обрабатываются только события подключения/отключения удаленного
 доступа. Суммируется общее время подключения для всех соединений с одинаковым
 ИМЕНЕМ СОЕДИНЕНИЯ и ЛОГИНОМ. В конце полученные суммы переводятся из секунд в
 дни:часы:минуты:секунды и выводятся в компонент TListView. Записи журнала
 обрабатываются в обратном порядке и поэтому записи на отключение предшествуют
 записям на подключение, т.е. записи на отключение - нечетные, а записи на
 подключение - четные. Программа будет работать правильно даже в случае, если
 встретится ситуация когда подключение было, а отключения нет. В программу
 введена специальная константа SERVER_DELAY_DISCONNECT, значение ее получено
 опытным путем и для другого провайдера может быть другим.
 Программа может использоваться как шаблон для написания других программ.
 При написании программы использовались: статья с "Королевства Delphi",
 пример с MSDN, файл winnt.h из Borland C++, файлы справки по Win32 API и
 Delphi7.
 Программа проверялась в Windows XP (Delphi7 и Turbo Delphi 2006 for Win32).
